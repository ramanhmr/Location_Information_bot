Модуль "Bot" содержит в себе телеграмм-бота.
  Имя бота: Location_Information_bot
  Токен: 1720794432:AAHCQfcCsKnYU6GLvwgpBLcg6AScogcinG4
При получении ботом сообщения в формате "[Город], [показатель], [показатель]" отвечает сообщением с данными о искомых показателях для данного города из базы данных mySQL.
В качестве разделителя в сообшениях, получаемых ботом должна использоваться запятая. Первая часть сообщения воспринимается как название города.
Если в сообщении будет указано только название города бот выведет всю доступную о нём информацию.
Доступ к базе данных со стороны телеграмм-бота ограничен только чтением.


Моудль "restmysql" содержит в себе приложение по взаимодействию с базой данных mySQL через HTTP запросы.
Запросы при обращении к базе данных:
  / (get) - отображает всю базу данных
  / (post) - добавляет город, описанный в прилагаемом JSON файле в базу данных (JSON файл должен быть представлен в формате:
      {
      "name": "[название города]",
      "population": [численность населения],
      "country": "[название страны]"
      })
  /[id] (get) - отображает информацию о городе с соответствующим ID.
  /[id] (post) - изменяет информацию о городе с соответствующим ID в соответствии с прилагаемым JSON файлом.
  /[id] (delete) - удаляет из базы данных город с соответствующим ID.
  /name/[название] (get) - отображает информацию о городе с соответствующим названием.
  /name/[название] (post) - изменяет информацию о городе с соответствующим названием в соответствии с прилагаемым JSON файлом.
  /name/[название] (delete) - удаляет из базы данных город с соответствующим названием.
  
  
  База данных располагается на локальном сервере mySQL по адресу "http://localhost:8080/city_info/"
